# Naming Convention

## `workItemRef`

- Format: `<PREFIX>-<digits>`
- Exemples: `GH-123`, `PDEV-456`

## Slug

- minuscule, `kebab-case`
- court, lisible, orienté objectif

## Fichiers

- agents: `<agent>.md`
- commandes: `<command>.md`
- skills: `<skill>/SKILL.md`
- templates: `*.template.md|yaml`

## IDs dans spec

- `F-`, `AC-`, `API-`, `EVT-`, `DM-`, `NFR-`, `RSK-`, `OQ-`, `DEC-`
